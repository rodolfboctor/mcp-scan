## v2 Requirements (v2.0.1) - Security Platform Upgrade

### Validated (Phases 1-8)

#### Coverage Breadth
- [x] **COV-01**: Detect Zed config at `~/.config/zed/settings.json`.

- [x] **COV-03**: Detect Cline config with glob support for versioned extension folders.
- [x] **COV-04**: Detect Roo Code config with glob support for versioned extension folders.
- [x] **COV-05**: Detect Amp config at `~/.amp/config.json`.
- [x] **COV-06**: Detect Plandex config at `~/.plandex/config.json`.
- [x] **COV-07**: Detect ChatGPT Desktop config (macOS) at `~/Library/Application Support/com.openai.chat/settings.json`.
- [x] **COV-08**: Detect GitHub Copilot config at `~/.config/github-copilot/apps.json`.
- [x] **COV-09**: Robust support for `.mcp.json` at project root.

- [x] **CI-01**: `--ci` flag for `scan` command that exits 1 on any findings (Critical to Info).

#### Security & Compliance
- [x] **SEC-01**: Pattern-based Data Flow Analysis (clipboard, DB, process sinks).
- [x] **SEC-02**: Network Egress Monitoring with hex/reversed URL detection.
- [x] **SEC-03**: Privacy Engine with PII detection and Risk Scoring.
- [x] **SEC-04**: Compliance Mapping for SOC 2, GDPR, HIPAA, PCI-DSS, NIST.
- [x] **SEC-05**: Full SARIF v2.1.0 export and GitHub Action integration.
- [x] **SEC-06**: Granular Policy Engine (skip, block, warn, override).
- [x] **SEC-07**: SBOM Generation (SPDX & CycloneDX v1.5).

#### Documentation
- [x] **DOC-03**: v2.0 Documentation refresh and launch announcements.
