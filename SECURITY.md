# Security Policy

mcp-scan is a security tool and we take your security seriously.

## Vulnerability reporting

If you discover a security vulnerability, please send an email to **abanoub.rodolf@gmail.com**.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- A suggested fix if you have one.

We will acknowledge your report within 48 hours. For critical vulnerabilities, we aim to release a fix within 14 days. We coordinate disclosure once a fix is available. Please do not open public issues for security vulnerabilities.

## Supported versions

| Version | Supported |
|:--------|:----------|
| Latest  | Yes       |
| < 1.0.0 | No        |

## Scope

- mcp-scan CLI tool.
- mcp-scan npm package.
- rodolfboctor/mcp-scan GitHub Action.

## Out of scope

- Issues in scanned MCP servers. Please report those to the server maintainer.
- Social engineering attacks.
- Vulnerabilities requiring physical access to the target machine.

## Disclosure format

Fixed vulnerabilities are documented with advisory IDs in the format MCP-SCAN-YYYY-NNN.
